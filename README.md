# Commands

```git pull``` - retrieve the latest changes

```git branch lab_X```

```git checkout lab_X```

_or_

```git checkout -b lab_X``` - move to a new branch

```git add``` - add __all__ the changed files

```git status```

```git commit -m "added lab_X"```

```git push -u origin lab_X```
